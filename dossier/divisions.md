# ATMOS // UNRAW : Divisional Breakdowns

The ATMOS ecosystem is partitioned into distinct functional divisions, each governed by its own technical and aesthetic parameters.

## 01. MUSIC DIVISION // VISUALS
*Telematic Transmission & Cinematic Immersion*

The Music Division serves as the emotional and rhythmic anchor of the ecosystem. It is managed via a live telematic HUD that integrates visual nodes with high-fidelity navigation.

- **Nodal Navigation**: Users cycle through the Visual MV library using a decentralized grid model. 
- **The Obsidian Control Pill**: A minimalist playback interface that anchors the viewport, featuring "Stealth Mode" auto-hide logic for 100% focus.
- **Active Visuals**:
    - `Backseat`: Official MV transmission.
    - `Sugarcoat`: Creative quadrant visual node.
    - `Baby, Not Baby`: Narrative system calibration.

## 02. APPAREL DIVISION // KINETIC
*Functional Lifestyle Systems*

The Apparel Division focuses on "Kinetic Gear"—garments designed for the architectural transition between digital and physical environments. We move away from fast-fashion toward "Capsule Systems."

- **Capsule A // System 01**:
    - *Tactical Street Cargos*: Heavyweight cotton ripstop with Y2K architectural fit.
    - *Racing Shield Windbreaker*: Technical shell with reflective piping and water-resistance.
    - *Classic Indigo Trucker*: 14oz Japanese Selvedge denim with reinforced tailoring.
- **Capsule A // System 02**:
    - *Peplum Denim Corset*: Avant-garde editorial silhouette with raw-edge detailing.

## 03. DIGITAL DIVISION // ARCHITECTURE
*High-Fidelity Codebase*

The Digital Division is the platform itself—an architectural web application engineered for the modern era.

- **The Stack**: React 19, TypeScript, Vite, Framer Motion, Lenis Smooth Scroll.
- **Design Statement**: High-fidelity interactive physics and momentum-based navigation create an editorial "Lookbook" feel that transcends traditional e-commerce.
