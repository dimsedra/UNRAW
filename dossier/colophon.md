# ATMOS // UNRAW : Technical Colophon

The digital infrastructure supporting the ATMOS ecosystem is as foundational as the aesthetic itself. This colophon documents the "Machine" behind the brand.

## 01. CORE STACK
*Production-Grade React Foundation*

- **Framework**: React 19 (using strict state management and `useLayoutEffect` for physics).
- **Runtime**: Node.js v20.x [LTS] / Vite 8.0 [Modular Build System].
- **Language**: TypeScript 6.0 [Strict Mode].
- **Physics**: Framer Motion 12.0 [Layout Projections & Spring Systems].
- **Smooth Scroll**: Lenis 1.3 [High-Fidelity Momentum Engine].

## 02. DESIGN TOKENS

- **Primary Canvas**: `#0A0A0A` [Obsidian Dark].
- **Accent Highlighting**: `#00FF41` [Telematic Green] / `#FFFFFF` [Optic White].
- **Grid System**: Responsive Safe-Zone Grid [SZG] based on architectural modular units.
- **Typography**: 
    - *Headlines*: Modernist Sans-Serif [Inter / Roboto / System Sans].
    - *Metadata*: Monospaced [JetBrains Mono / Source Code Pro].

## 03. SYSTEM ASSETS

- **Visual Viewport**: Integrated `react-youtube` API with custom state-driven chapter management.
- **Interactive Layers**: 
    - *Dynamic Island Search*: Derived-state island navigation in the Apparel Division.
    - *Cinematic Overlay*: Fixed-position Obsidian Control Pill with 3s auto-fade logic.

## 04. DEVELOPMENT PHILOSOPHY

Developed using an **Atomic Methodology**. Components are designed as modular architectural units, ensuring that the system can scale without compromising "System Calm" or structural integrity.
