# ATMOS Roadmap: Future System Updates

This document outlines the planned evolutionary steps for the ATMOS digital ecosystem, focusing on architectural expansion, utility, and visual media integration.

## 1. Atmospheric Video Overlay (YouTube Integration)

- **Trigger Logic:** The sidebar's Music Division play button will serve as the gateway to the visual layer.
- **Overlay Architecture:**
  - Clicking 'Play' will trigger a YouTube Video Player overlay that occupies the current content zone.
  - The player remains constrained by the persistent Sidebar and Top-Bar, maintaining the "System Shell" feel.
- **Mobile Adaptation:** On smaller screens where containment would compromise video clarity, the player will transition to a full-page overlay (optimized for standard video aspect ratios).
- **Navigation Controls:** The player will include `< >` navigation elements to cycle between different Atmos/UNRAW visual experiences (MVs, song chapters) within the same overlay context.

## 2. Expansion: Roster Architecture

- **Branding Shift:** The `UNRAW` top-bar link will be renamed to `Roster` to accommodate future acts/talent.
- **Roster Hub UI:**
  - Implementation of a dedicated Roster index page.
  - **Visual Inspiration:** A Spotify-style artist directory layout.
  - **Components:** High-fidelity circular profile imagery paired with minimalist typography for each artist/group.
- **Trigger:** This transition will activate once multiple creative acts are ready for deployment within the ecosystem.
