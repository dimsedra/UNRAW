# ATMOS // UNRAW
### A High-Fidelity Digital Ecosystem & Architectural Showcase

![ATMOS Banner](https://raw.githubusercontent.com/user-attachments/assets/atmos-placeholder-banner.png)

> "Rejecting traditional consumption cycles. Developing environments to sustain a singular, prolonged feeling. The identity evolves naturally—stripping excess until only function remains."

**ATMOS // UNRAW** is a professional-grade architectural web application designed to demonstrate the marriage of high-fidelity aesthetics with technical precision. Moving away from standard portfolio tropes, this project introduces a "Technical Viewport" philosophy—where every interaction is intentional, and every visual is curated.

---

## ⚡ Technical Core

Built on a foundation of performance and smooth interactive physics:

- **React 19 + TypeScript**: Type-safe development with the latest React feature set.
- **Vite**: Ultra-fast build tool and development server.
- **Framer Motion**: Powering state-of-the-art layout transitions and micro-animations.
- **Lenis Smooth Scroll**: Implementing high-end momentum scrolling for an editorial navigation feel.
- **Atmospheric Video Engine**: A custom viewport layer built on `react-youtube` for precise chapter navigation and cinematic immersion.

---

## 🌗 Key Features

### 1. The Command Sidebar (Music Division)
A live telematic HUD that acts as the system's structural anchor. It features:
- **Interactive Marquee**: Dynamic title management that triggers only on hover to preserve "System Calm."
- **Nodal Navigation**: Rapid cycling through visual MVs directly from the sidebar grid.

### 2. Cinematic Video Overlay
A modern minimalist media viewport designed for 100% focus:
- **Obsidian Control Pill**: A floating, high-fidelity navigation island.
- **Stealth Mode**: UI auto-hide logic that fades controls away after 3 seconds of inactivity.
- **Takeover Logic**: Responsive system-takeover for maximum immersion on mobile devices.

### 3. Editorial Shop Experience
A curated product inventory designed as an architectural lookbook, moving away from generic e-commerce patterns toward a "Peer Alignment" model.

## 🛠️ Installation & Setup

Ensure you have [Node.js](https://nodejs.org/) installed (v18+ recommended).

1. **Clone the Repository**
   ```bash
   git clone https://github.com/your-username/atmos-unraw.git
   cd atmos-unraw
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment**
   Create a `.env.local` file in the root directory:
   ```bash
   VITE_GOOGLE_CLIENT_ID=your_client_id_here
   ```

4. **Launch Development Environment**
   ```bash
   npm run dev
   ```

---

## 📦 Production Deployment

To generate a highly optimized production bundle:

```bash
npm run build
```

The output will be located in the `dist/` directory, ready for deployment to any static hosting provider.
